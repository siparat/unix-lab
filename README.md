### Для работы проекта должны быть установлены:
- Node.js (версия 16 или выше)
- npm (менеджер пакетов, идет в комплекте с Node.js)

### Шаг 1
Склонируйте репозиторий
```bash
git clone https://github.com/siparat/unix-lab.git
```

### Шаг 2
Установите зависимости
```bash
cd unix-lab
npm i
```

### Шаг 3
Скопируйте файл config.example.json в config.json и настройте конфиг
``` bash
cp config.example.json config.json
```

### Шаг 4
Соберите проект
``` bash
npm run build
```

### Шаг 5
Запуск
``` bash
npm run start
```
